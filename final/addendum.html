JS Functionality Process

  1. Grid Generation and Initialization: I have looped through each cell for defining grid size. To generate HTML buttons with unique IDs and onclick events.
  2. I have created function s(x) for identifying the cell by ID 'X'.
  3. sc(row, col) Function will be triggered when a cell is clicked and its adjacent cells. After toggling the cells, it checks the solve function if the game is completed.
  4. solveEnd() Function is similar to solve() function but it resets the game and timer irrespective of the game win or loss.
  5. changeColor() Function will be updating the colors based on the cells innertext. It also counts the color of yellow lights for checking win condition
  6. Randomization() Function will create random grid cells and included with solve function inside it.
  7. Reset() function will reset the timer and reset the grid by randomizing it.
  8. Restart confirmation dialog will help to confirm for restarting the game.
  1. I implemented buttons styling for visual appealing
  2. And also styled hover to increase visual enhancements.
  3. Timer styling for font size and color.

    The Challenges I faced while implementing Lights Out Game:

  1. Grid Interaction, As each cell shoukd be toggled and if one cell is toggled other adjacent cells should also react.
  2. Also, I felt little challenging while making winning possibilities, Because sometimes grid cells will be in uncertain possibility to win making it infinite game play.
  3. Later, I found few patterns, With these patterns the game would complete succesfully. With the help of these patterns randomizing game logic became easy.
  4. I, also ensured game logic by testing and debugging under many scenarios. I have provided hint option to check if bottom lights are left out in that pattern then look out for top row hints.
